# Churn Modelling using Artificial Neural Networks. 

The key features used in the dataset are:
- CreditScore
- Geography
- Age
- Tenure
- Balance
- NumOfProducts
- HasCreditCard
- IsActiveMember
- EstimatedSalary
- Exited

Using these features, the model predicts which users are more likely to leave the bank.
